The HTML <head> Tag and Its Elements

Common Elements Inside <head>

  • <title>: Sets the page title (shown in browser tab).
  • <meta charset="utf-8">: Sets the character encoding.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Makes your site mobile-friendly.
  • <meta name="description" content="A short summary of your page.">: Helps with SEO.
  • <meta name="keywords" content="html, tutorial, web">: (Less important now, but still used.)
  • <link rel="stylesheet" href="style.css">: Links to external CSS files.
  • <link rel="icon" href="favicon.ico">: Sets the browser tab icon.
  • <script src="script.js"></script>: Links to JavaScript files.
  • <base href="https://example.com/">: Sets the base URL for all relative links.

Example <head> Section

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Webpage</title>
    <meta name="description" content="A beginner HTML page.">
    <link rel="stylesheet" href="style.css">
    <link rel="icon" href="favicon.ico">
    <script src="main.js"></script>
</head>

Why is the <head> Important?

  • Controls how your page appears in search engines and on social media.
  • Links to stylesheets and scripts for design and interactivity.
  • Sets the page title and icon for a better user experience.
